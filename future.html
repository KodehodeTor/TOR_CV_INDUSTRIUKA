Fremtidsutsikter

Denne siden forklarer hvor jeg er på vei hen — kortsiktige mål, fremtidens drømmer og stegene jeg akter å utføre for å nå dem.

Kortsiktige mål (neste 1–2 år)

  • Fullføre Full-Stack kurs via Kodehode
  • Mestre HTML, CSS og JavaScript
  • Få en junior posisjon i et firma eller fortsette Full-Stack på høyskolen.
  • Medvirke i et open-source prosjekt på GitHub

Fremtidens drømmer

I det lengre løp ønsker jeg å ha nok verktøy i kassa mi til å skape noe eget som bidrar til andre og utgjør en positiv forskjell for dem. Et digitalt verktøy som får oss til å se oss selv og ikke kun pikslene på skjermen. Et sosial filosofiskt verktøy som kan gi en mulighet til å lære selv realisering.

Lærings veikart

Akkurat nå
HTML, CSS, JavaScript
Neste
Figma, REACT
Etter det
Grunnleggende back-end (Node.js)
Etter hvert
Databaser, API design, cloud deployment

Karrierer jeg finner interessante

Front-End Utvikler

Bygge brukervennlige grensesnitt som er raske, tilgjengelig og lett å bruke.

UX / Product Designer

Jeg har en historikk med tegning og kunst. Selvom dette er en side jeg har utfordringer å dykke inn i på kommando er det interessant med kombinasjonen av teknologi og design.

Full-Stack Developer

Dette er min største drøm, det å leve på begge sider av stacken og være komfertabel med det. Bygge både hva brukere ser og systemene som støtter i bakhånd.
